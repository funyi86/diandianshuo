# 点点说 · 图卡沟通板

给幼儿使用的辅助沟通（AAC）网页应用：点图卡朗读词语、组句表达。支持照片图卡、真人录音、离线使用（PWA）。

在线地址：https://funyi86.github.io/diandianshuo/

- 单文件前端，无后端；所有照片、录音、自定义词卡存储在设备本地（IndexedDB / localStorage），不上传。
- 添加到手机主屏幕后可完全离线使用。
- 家长设置：长按右上角 🔒 1.5 秒。
